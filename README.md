# 🐟 Fish Detection using YOLOv8

A web application for fish detection using a custom-trained YOLOv8 model and Streamlit.

## Features

- Upload an image
- Detect fish automatically
- Display bounding boxes
- Show confidence scores

## Technologies

- Python
- YOLOv8 (Ultralytics)
- Streamlit
- Pillow

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Model

The application uses a custom-trained YOLOv8 model (`best.pt`).
